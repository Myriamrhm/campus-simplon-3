JS - Les Simploniens

Un petit bout de code fun pour lister des simploniens et les faire agir ; )

Créer un tableau d'objets. Chaque objet représente un simplonien. Chaque simplonien a les propriétés name (str), age (num) et gimmick (str).

Parcourir les simploniens avec une boucle (for ou forEach de préférence). Afficher chaque simplonien dans la console.

Utiliser les deux méthodes ci-dessous (Voir MDN):
document.createElement
element.appendChild
Pour créer autant de li qu'il y 'a de simploniens... :D
Puis les insérer dans une liste : D

Au click sur chaque simplonien : afficher son gimmick dans la console !!!
