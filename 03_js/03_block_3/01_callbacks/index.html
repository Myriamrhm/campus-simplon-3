Les callbacks

un callback est une fonction, argument d'une autre fonction.
Pour rappel, en JS, les fonctions sont des valeurs comme les autres.
On peut donc les assigner à une variable OU, dans le cas de callbacks, les passer comme paramètre d'une autre fonction.
Le but de la manoeuvre est de pouvoir exécuter le callback et retourner une valeur dans le scope de la fonction hôte.
C'est très utile dans les cas de programmation asynchrone (AJAX, Promesses, Opérations sur fichiers volumineux)
