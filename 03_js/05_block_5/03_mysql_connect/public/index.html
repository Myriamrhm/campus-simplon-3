JS - Node Connect Mysql

1 - GET

Connecter votre serveur node à mysql.
Utiliser ajax pour requêter le serveur.
Récupérer une liste de résultat : par exemple tous les users de la table user.
Faire l'affichage HTML sous forme de liste OU de table.

2 - POST

Créer un formulaire pour saisir les informations d'un user.
Au click sur un boutton, envoyer les données en POST vers le serveur node via AJAX.
Votre application express récupère le post.
Passer ensuite les données à la méthode appropriée de votre fichier database.js
